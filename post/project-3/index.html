Introduction

The advanced progress in technology has significantly affect nowadays life, including our ways and considerations in choosing something. Usually, when we want to buy food, drinks, or any other item via an online application, we will get recommendations from the app that make us consider buying them. The feature that generates recommendations for us is called the recommendation system. Nowadays, the recommendation system has become one of the most important assets for the majority of big companies, as it is quite proven to increase the company’s revenue.

In this project, I tried to create a recommendation system based on the article’s content. The recommendation system will generate a list of top 5 articles related to the current article as the output. The data used for this project was the mini version of the “CNN News Articles from 2011 to 2022” dataset from Kaggle. This dataset contains 9 columns (keywords, headline, description, etc.) and around 4000 rows (articles). I only used the “description” column because the recommender system in this project is content based. The dataset was then preprocessed, vectorized using TF-IDF vectorizer, and fitted to NMF model.

Link to the dataset: https://www.kaggle.com/datasets/hadasu92/cnn-articles-after-basic-cleaning
